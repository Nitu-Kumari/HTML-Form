# What is an event what is an event listener?
~~~
An event listener is a function in a computer program that waits for an event to occur;
 event may be a user clicking or moving the mouse, pressing a key on the keyboard, or an internal timer or interrupt. 
 The listener is in effect a loop that is programmed to react to an input or signal.
